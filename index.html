My Journey In Tech

I started programming using an online course called "CS50 by Harvard University." The programmes I built in C as part of this course include one that assigns a reading age to a text file, another which applies filters to images, and one which recovers images from a corrupted memory file. Eventually, we moved onto Python and SQL where I built a program to read DNA as text and compare the results to a database of people to find a match.

Following this, I started an intensive youth course with Generation where I studied to become a Data Engineer. The Python we coded with was more advanced and the applications built were multifaceted and included tools such as MySQL and Jupyter Notebooks. This was also my introduction to cloud services such as AWS and Google Cloud. Using AWS I alongside my team built an ETL pipeline for a large cafe with 118 branches. We transformed our clients approach to data and how they use data to benefit their company
